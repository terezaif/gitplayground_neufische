# gitplayground_neufische

##
Requirements:
- Python: 3.8.5

## Setup
Having Anaconda installed then create your ENV with

```bash
conda env create -n ds-playground -f environment.yml
```

With pyenv installed

```bash
pyenv local 3.8.5
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
